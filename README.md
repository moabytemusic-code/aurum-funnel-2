# Get AI Finance — Onboarding Funnel

A streamlined onboarding funnel for warm prospects interested in the Get AI Finance Digital Banking Ecosystem.

## Features

- **Progress Tracking** - Visual progress indicators throughout the funnel
- **Embedded Video** - Keep users on-page with integrated presentation
- **Multi-Step Application** - Clean, frictionless form with progress saving
- **Mobile Responsive** - Optimized for all devices
- **FAQ Section** - Expandable answers to common objections
- **Support Integration** - Multiple contact options for assistance

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Deployment:** Vercel

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the funnel.

### Build

```bash
npm run build
npm start
```

## Customization

### Video Embed
Replace `YOUR_VIDEO_ID` in `src/app/page.tsx` with your actual YouTube/Vimeo video ID.

### Application Backend
Connect the form submission in `src/app/application/page.tsx` to your CRM or backend API.

### Contact Information
Update support email and booking links in the footer and support sections.

## Deployment

Deploy to Vercel:

```bash
vercel
```

Or connect your GitHub repo to Vercel for automatic deployments.
